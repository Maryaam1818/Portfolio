Internal HR
Management System

TMKiiN Desktop App is an internal HR management tool built in Java for a Tunisian software company connecting employees, HR managers, admins, and secretaries in one unified desktop application. Staff can submit leave requests, track approval status, and manage their profiles, while managers handle accounts and HR workflows from their own dedicated spaces. Built solo during an internship, the challenge was designing a multi-role authentication system with four distinct user interfaces from scratch using Java Swing, Eclipse, and JDBC in just a few weeks.

Overview

The
challenge

TMKiiN, a Tunisian software company, had no internal system connecting its employees, HR managers, admins, and secretaries. Leave requests were handled manually, account management was scattered, and there was no centralized way for staff to communicate or track requests.

Built solo during an internship, the app was designed around four distinct user roles from day one each with its own login and interface. Java was chosen for its robust GUI capabilities via Swing, Eclipse as the IDE, and JDBC for database connectivity.

Delivered a fully functional multi-role desktop application covering authentication, leave requests, approvals, and account management within a few weeks..
